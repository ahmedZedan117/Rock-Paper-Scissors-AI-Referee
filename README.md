# Rock-Paper-Scissors AI Referee
This repository contains an intelligent computer vision system built to act as an automated referee for the classic Rock-Paper-Scissors game. By leveraging deep learning and real-time hand gesture recognition, the model accurately detects and classifies hand signs. Whether judging a match between two players or interacting directly with a user, this project demonstrates practical, real-world applications of computer vision in interactive systems
